Tablas HTML

Objetivos de aprendizaje

  • Comprender la estructura básica de una tabla HTML
  • Crear tablas con filas, columnas y celdas
  • Implementar encabezados de tabla para accesibilidad
  • Combinar celdas usando rowspan y colspan
  • Usar elementos semánticos como thead, tbody y tfoot
  • Crear tablas complejas y estructuradas

Organizando datos con tablas

Las tablas son esenciales para presentar datos estructurados en la web. Permiten organizar información en filas y columnas, facilitando la lectura y comparación de datos numéricos, horarios, precios y más.

Estructura básica: <table>, <tr>, <td>

Una tabla HTML se construye con tres elementos fundamentales:

<table>
  • Contenedor principal
  • Define la tabla
  • Encierra todos los elementos
<tr>
  • Table Row (fila)
  • Contiene las celdas
  • Una por cada fila
<td>
  • Table Data (celda)
  • Contiene los datos
  • Dentro de <tr>
Sintaxis básica de tabla HTML 
<table>
  <tr>
    <td>Celda 1</td>
    <td>Celda 2</td>
  </tr>
  <tr>
    <td>Celda 3</td>
    <td>Celda 4</td>
  </tr>
</table>

Tip: El atributo border="1" en la etiqueta <table> muestra los bordes de la tabla durante el desarrollo, pero en producción se usan estilos CSS.

Encabezados de tabla: <th>

Los encabezados (<th>) identifican el contenido de las columnas o filas. Son esenciales para la accesibilidad y mejoran la legibilidad.

Nombre Edad Ciudad
María 25 Madrid
Carlos 30 Barcelona
Tabla con encabezados HTML 
<table>
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
    <th>Ciudad</th>
  </tr>
  <tr>
    <td>María</td>
    <td>25</td>
    <td>Madrid</td>
  </tr>
</table>

Importancia de los encabezados

Los lectores de pantalla usan los encabezados para navegar las tablas. Siempre usa <th> para encabezados y <td> para datos.

Combinando celdas: colspan y rowspan

Estos atributos permiten que una celda ocupe múltiples columnas o filas, útil para encabezados agrupados o celdas especiales.

Información Personal Contacto
Nombre Apellido correo@ejemplo.com
María García
Tabla con celdas combinadas HTML 
<table>
  <tr>
    <th colspan="2">Información Personal</th>
    <th>Contacto</th>
  </tr>
  <tr>
    <td>Nombre</td>
    <td>Apellido</td>
    <td rowspan="2">correo@ejemplo.com</td>
  </tr>
  <tr>
    <td>María</td>
    <td>García</td>
  </tr>
</table>

¡Planifica cuidadosamente!

Al usar rowspan y colspan, asegúrate de que el número total de celdas en cada fila sea consistente. Un error común es crear filas con diferente número de celdas.

Estructura semántica: <thead>, <tbody>, <tfoot>

Estos elementos agrupan las filas de una tabla, mejorando la accesibilidad, el estilo y la impresión.

<thead>

Contiene las filas de encabezado (cabecera de la tabla).

<thead>
  <tr>
    <th>Producto</th>
    <th>Precio</th>
  </tr>
</thead>

<tbody>

Contiene el cuerpo principal de datos de la tabla.

<tbody>
  <tr>
    <td>Laptop</td>
    <td>$999</td>
  </tr>
</tbody>

<tfoot>

Contiene las filas de pie (totales, resúmenes).

<tfoot>
  <tr>
    <td>Total</td>
    <td>$1998</td>
  </tr>
</tfoot>
Tabla completa con estructura semántica HTML 
<table>
  <caption>Lista de Productos</caption>
  <thead>
    <tr>
      <th>Producto</th>
      <th>Precio</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Laptop</td>
      <td>$999</td>
    </tr>
    <tr>
      <td>Mouse</td>
      <td>$25</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Total</td>
      <td>$1024</td>
    </tr>
  </tfoot>
</table>

Títulos de tabla: <caption>

El elemento <caption> proporciona un título descriptivo para la tabla. Es importante para la accesibilidad y ayuda a los usuarios a entender el contenido.

Horario de Clases - Primer Semestre
Hora Lunes Martes
8:00-9:00 Matemáticas Español
Tabla con caption HTML 
<table>
  <caption>Horario de Clases - Primer Semestre</caption>
  <!-- resto de la tabla -->
</table>

Posicionamiento del caption

El <caption> debe ser el primer elemento dentro de <table>. Puede posicionarse arriba o abajo de la tabla usando CSS (caption-side: top|bottom).

Tablas anidadas y usos apropiados

¿Cuándo usar tablas?

USAR tablas para:
  • Datos tabulares
  • Horarios y calendarios
  • Comparación de productos
  • Resultados deportivos
  • Datos financieros
NO USAR tablas para:
  • Layout de página
  • Posicionar elementos
  • Diseño de formularios
  • Menús de navegación
  • Estructura general del sitio

Lección histórica

En los primeros días de la web (1990s-2000s), las tablas se usaban para crear layouts completos. Esto ya no es recomendado porque crea problemas de accesibilidad, mantenimiento y no es responsive. Hoy usamos CSS Grid y Flexbox para layouts.

Buenas prácticas para tablas

Accesibilidad

  • Usa <th> para encabezados
  • Agrega <caption> descriptivo
  • Usa scope="col" o scope="row"
  • Evita tablas anidadas complejas

Responsive

  • Usa <div class="table-responsive">
  • Considera mostrar datos verticalmente en móviles
  • Evita tablas muy anchas (más de 6-7 columnas)
  • Usa overflow-x: auto para scroll horizontal

Código limpio

  • Usa indentación consistente
  • Agrupa con <thead>, <tbody>, <tfoot>
  • Comenta secciones complejas
  • Mantén los nombres de clases semánticos

Regla general

Si los datos son tabulares, usa una tabla. Si estás creando layout visual, usa CSS. Las tablas son para datos, no para diseño.

Actividad práctica: Crea tu horario de clases

Es hora de aplicar lo aprendido creando tu propio horario de clases usando solo HTML.

  1. Crea un archivo llamado horario.html con la estructura básica: 
    <!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi Horario de Clases</title>
</head>
<body>
  <h1>Mi Horario de Clases</h1>
  <!-- Tu tabla irá aquí -->
</body>
</html>
  2. Crea una tabla con las siguientes características:
    • Usa <table border="1"> para ver los bordes
    • Agrega un <caption> con "Mi Horario Semanal"
    • Encabezados para los días de la semana (Lunes a Viernes)
    • Al menos 5 filas con horarios de 7:00 a 12:00
    • Una fila con colspan="5" para el RECESO
    • Pon tus materias reales o inventadas
  3. Mejora tu tabla (opcional):
    • Agrega <thead>, <tbody> y <tfoot>
    • En el <tfoot>, añade el total de horas semanales
    • Usa rowspan para una clase que dure dos horas seguidas
    • Agrega una columna para el sábado si tienes clases ese día
  4. Revisa tu trabajo:
    • ¿Todos los <tr> tienen el mismo número de celdas?
    • ¿Los encabezados usan <th> en lugar de <td>?
    • ¿La tabla se ve bien al abrirla en el navegador?
    • ¿El código está correctamente indentado?

Consejo profesional

Empieza con una tabla simple de 2x2 para asegurarte de que la estructura básica funciona, luego expande a tu horario completo. Usa comentarios HTML (<!-- comentario -->) para organizar las diferentes partes de tu tabla.

Resumen de la lección

Las tablas HTML (<table>) organizan datos en filas (<tr>) y columnas. Las celdas de datos usan <td> y los encabezados usan <th>.

Los atributos colspan y rowspan permiten combinar celdas horizontal o verticalmente. Los elementos semánticos <thead>, <tbody>, y <tfoot> mejoran la estructura y accesibilidad.

Siempre usa <caption> para describir la tabla y considera la accesibilidad usando scope en los encabezados. Recuerda: las tablas son para datos tabulares, no para diseño de layouts.

Con estos conocimientos, puedes crear desde simples listas de precios hasta complejos horarios y calendarios, todo mientras mantienes tu código accesible y semánticamente correcto.
