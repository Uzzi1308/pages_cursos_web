Listas y enlaces HTML

Objetivos de aprendizaje

  • Crear listas ordenadas y no ordenadas
  • Anidar listas para estructurar contenido complejo
  • Enlazar páginas internas y sitios externos

  • Crear menús de navegación funcionales

Organizando contenido con listas

Las listas son fundamentales para organizar información en la web. Piensa en menús de navegación, índices, pasos de un proceso o cualquier contenido que tenga elementos relacionados.

Listas no ordenadas <ul>

Se usan cuando el orden de los elementos no importa. Ideal para:

Lista de compras
  • Leche
  • Pan
  • Huevos
  • Fruta
Características
  • Fácil de leer
  • Sin orden específico
  • Usa viñetas (bullets)
  • Para agrupaciones
Sintaxis de lista no ordenada HTML 
<ul>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
  <li>Tercer elemento</li>
</ul>

Tip: Cada <li> debe estar dentro de <ul> o <ol>. Los elementos de lista pueden contener texto, imágenes, enlaces y hasta otras listas.

Listas ordenadas <ol>

Se usan cuando el orden SÍ importa. Perfectas para:

  • Recetas de cocina
  • Instrucciones paso a paso
  • Rankings o clasificaciones
  • Tutoriales secuenciales
    • Ejemplo: Pasos para hacer café HTML 
    <ol>
  <li>Calentar agua</li>
  <li>Poner café en el filtro</li>
  <li>Verter agua caliente</li>
  <li>Dejar reposar 3 minutos</li>
  <li>Servir y disfrutar</li>
</ol>

    Lista ordenada en navegador

    1. Calentar agua
    2. Poner café en el filtro
    3. Verter agua caliente
    4. Dejar reposar 3 minutos
    5. Servir y disfrutar

    El poder de los enlaces: <a>

    Los enlaces (hyperlinks) son lo que hace a la web "web". Conectan páginas, recursos y sitios, creando la red global de información.

    El elemento más importante de HTML

    Hipertexto
    La "H" en HTML significa HyperText. Los enlaces son la esencia misma de la web.
    Navegación
    Sin enlaces, las páginas serían islas aisladas. Los enlaces crean caminos entre contenido.
    Estructura
    Los menús de navegación son listas de enlaces. Organizan el acceso a todo tu sitio.

    Sintaxis básica de enlaces

    Estructura del elemento <a> HTML 
    <a href="URL">Texto del enlace</a>

    El atributo href (hypertext reference) es obligatorio. Indica hacia dónde lleva el enlace.

    Enlace interno
    href="#seccion"
    Enlace externo
    target="_blank"
    Enlace email
    href="mailto:"
    Enlace teléfono
    href="tel:"

    Texto descriptivo

    El texto del enlace debe describir claramente hacia dónde lleva. Evita textos genéricos como "haz clic aquí" o "más información".

    Mal: Haz clic aquí para ver nuestro catálogo.
    Bien: Explora nuestro catálogo de productos.

    Rutas: ¿Absolutas o relativas?

    https://misitio.com/proyectos/index.html
    Accede a un recurso desde cualquier lugar usando la URL completa.

    Estructura de ejemplo:

    sitio-web/
    index.html
    nosotros/
    index.html
    equipo.html
    proyectos/
    index.html
    web.html
    assets/
    estilos.css
    script.js

    ¿Cuándo usar cada tipo?

    • Absolutas: Para enlaces a otros sitios web
    • Relativas (raíz): Desde la raíz del sitio (comienzan con /)
    • Relativas (actual): Desde la ubicación actual del archivo
    • ../: Para subir un nivel en la estructura de carpetas

    Atributos que potencian tus enlaces

    target="_blank"

    Abre el enlace en una nueva pestaña o ventana.

    <a href="https://ejemplo.com" target="_blank">
  Abrir en nueva pestaña
</a>

    Usa con cuidado: Abrir muchas pestañas puede molestar a los usuarios. Recomendado solo para enlaces que deben mantener la página actual abierta.

    title="Descripción"

    Proporciona información adicional sobre el enlace.

    <a href="documento.pdf" title="Descargar PDF (2MB)">
  Descargar guía
</a>

    Aparece como tooltip al pasar el mouse. Mejora la accesibilidad y ayuda a los usuarios a entender hacia dónde va el enlace.

    Enlaces a email y teléfono

    Especiales para contacto directo.

    <!-- Enlace de correo -->
<a href="mailto:info@ejemplo.com">
  Envíanos un correo
</a>

<!-- Enlace de teléfono -->
<a href="tel:+1234567890">
  Llámanos
</a>

    En dispositivos móviles, estos enlaces abren la app de correo o marcan el número automáticamente.

    Creando menús de navegación

    Los menús de navegación son esenciales en cualquier sitio web. Combinan listas no ordenadas con enlaces para crear una estructura clara y accesible.

    Inicio
    Nosotros
    Servicios
    Portafolio
    Contacto
    Código del menú de navegación HTML 
    <nav>
  <ul>
    <li><a href="index.html">Inicio</a></li>
    <li><a href="nosotros.html">Nosotros</a></li>
    <li><a href="servicios.html">Servicios</a></li>
    <li><a href="portafolio.html">Portafolio</a></li>
    <li><a href="contacto.html">Contacto</a></li>
  </ul>
</nav>

    Buenas prácticas para menús

    • Usa la etiqueta semántica <nav>
    • Mantén el menú simple y claro (5-7 elementos máximo)
    • Organiza los enlaces de más importante a menos
    • Usa texto descriptivo para cada enlace
    • Asegúrate de que todos los enlaces funcionen

    Navegación dentro de la misma página

    Los enlaces internos (también llamados anclas) permiten saltar a diferentes secciones de la misma página. Perfectos para:

    • Índices de contenido
    • Botones "volver arriba"
    • Navegación entre secciones largas
    • FAQs con preguntas y respuestas
    Ejemplo de enlaces internos HTML 
    <!-- Enlace que apunta a una sección -->
<a href="#conclusion">Ir a la conclusión</a>

<!-- Sección destino con id -->
<h2 id="conclusion">Conclusión</h2>
<p>Contenido de la sección...</p>

<!-- Enlace para volver arriba -->
<a href="#top">Volver al inicio</a>

    El atributo id es clave

    Para que un enlace interno funcione, el elemento destino debe tener un id único. El enlace usa href="#id-del-elemento".

    Actividad práctica: Crea tu primer sitio web estructurado

    Es hora de aplicar todo lo aprendido creando un pequeño sitio web con múltiples páginas y menú de navegación:

    1. Crea una carpeta llamada mi-sitio-web con esta estructura:
      mi-sitio-web/
      index.html
      acerca-de/
      index.html
      proyectos/
      index.html
      contacto/
      index.html
    2. En index.html (página principal):
      • Crea un menú de navegación con enlaces a las otras páginas
      • Añade una lista ordenada con tus 3 pasatiempos favoritos
      • Añade una lista no ordenada con tus 5 películas/series favoritas
      • Incluye un enlace a tu sitio web favorito (usando target="_blank")
    3. En acerca-de/index.html:
      • Crea el mismo menú de navegación
      • Escribe un párrafo sobre ti
      • Crea una lista anidada con tus habilidades o cosas sobre ti
      • Añade un enlace de correo para contacto
    4. En proyectos/index.html:
      • Menú de navegación
      • Lista ordenada de proyectos personales
      • Enlaces internos para saltar entre secciones del proyecto
      • Agrega un titulo y un parrafo por cada proyecto y a los titulos ponles un ID unico que se usaran en los enlaces
    5. En contacto/index.html:
      • Menú de navegación
      • Enlace de teléfono (puede ser un número ficticio)
      • Enlace que descargue un archivo PDF (usa href="CV.pdf")
      • Enlace que vuelva a la página principal
    6. Reto extra: Crea un enlace en el pie de cada página que diga "Volver arriba" y funcione como enlace interno.

    Consejo profesional

    Mantén el menú de navegación idéntico en todas las páginas. Esto crea consistencia y ayuda a los usuarios a orientarse. Puedes copiar y pegar el código del menú en cada archivo.

    Resumen de la lección

    Las listas (<ul>, <ol>, <li>) organizan contenido relacionado. Las no ordenadas usan viñetas, las ordenadas usan números. Pueden anidarse para crear estructuras jerárquicas complejas.

    Los enlaces (<a href="URL">) conectan la web. El atributo href puede contener rutas absolutas (https://...), relativas (carpeta/archivo.html), o internas (#seccion).

    Atributos especiales como target="_blank" (abre en nueva pestaña), title="descripción" (tooltip), y esquemas como mailto: y tel: potencian la funcionalidad de los enlaces.

    Los menús de navegación combinan listas no ordenadas con enlaces dentro de la etiqueta semántica <nav>. Son esenciales para la usabilidad de cualquier sitio web.
