Introducción a HTML

¿Qué es HTML?

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje fundamental que define la estructura de todas las páginas web que existen en internet.

Puntos clave

  • HTML no es un lenguaje de programación, es un lenguaje de marcado
  • Un lenguaje de marcado usa etiquetas para organizar y estructurar información
  • HTML organiza el contenido mediante etiquetas que indican qué es cada elemento
  • Todo lo que ves en una página web comienza con HTML: textos, botones, imágenes, formularios
  • Sin HTML, simplemente no existiría la web como la conocemos

Analogía: HTML como estructura de una casa

HTML
La estructura y paredes de la casa
CSS
La pintura y decoración
JavaScript
La electricidad y funcionalidad

Importante

Aunque CSS (estilos) y JavaScript (funcionalidad) mejoran una página web, sin HTML nada de eso puede existir. HTML es siempre el punto de partida.

Contexto: ¿Cómo funciona la web?

Para entender HTML, primero necesitas comprender cómo funciona la web a nivel básico:

Paso 1
Un usuario escribe una dirección web (URL) en su navegador
Paso 2
El navegador solicita la página al servidor donde está alojada
Paso 3
El servidor responde enviando archivos HTML, CSS y JavaScript
Paso 4
El navegador interpreta el HTML y lo convierte en lo que el usuario ve en pantalla

Dato importante

Todo sitio web tiene un archivo inicial llamado index.html. Es la página principal que se carga por defecto cuando visitas un sitio.

Breve historia de HTML

HTML tiene una historia fascinante que vale la pena conocer:

1991
Tim Berners-Lee crea HTML en el CERN para compartir documentos científicos. La primera versión tenía solo 18 etiquetas básicas.
1995-1999
Surgen HTML 2.0, 3.2 y 4.01. La web comienza a popularizarse con mejoras en tablas, formularios y soporte básico para estilos.
2014
HTML5 se convierte en el estándar oficial. Trae etiquetas semánticas, soporte nativo para audio/video y mejoras enormes en estructura.
Actualidad
HTML5 es el estándar actual. Hoy existen más de 100 etiquetas y HTML sigue evolucionando con nuevas características.

Innovaciones de HTML5

  • Etiquetas semánticas: header, footer, nav, article, section, aside
  • Soporte nativo para audio y video sin necesidad de plugins
  • Canvas para dibujar gráficos y crear animaciones
  • Mejor estructura y SEO (Search Engine Optimization)
  • Geolocalización y almacenamiento local

¿Qué es SEO?

SEO significa Search Engine Optimization (Optimización para Motores de Búsqueda). Es el conjunto de técnicas para que tu sitio web aparezca mejor posicionado en Google y otros buscadores. HTML5 semántico ayuda a que los buscadores entiendan mejor tu contenido.

Herramientas que usaremos en el curso

Para trabajar con HTML necesitarás algunas herramientas básicas. No te preocupes, todas son gratuitas y fáciles de usar:

Visual Studio Code

Editor de código gratuito y potente. Es el más usado por desarrolladores. Ligero, con autocompletado inteligente y miles de extensiones útiles.

Google Chrome

Navegador web recomendado por sus excelentes herramientas de desarrollo (DevTools). Estándar en la industria para probar sitios web.

Chrome DevTools

Herramientas integradas en Chrome para inspeccionar elementos HTML, editar código en tiempo real, ver errores y depurar tu sitio web.

Extensiones recomendadas para VS Code

  • Live Server - Actualiza automáticamente tu navegador al guardar cambios
  • HTML CSS Support - Autocompletado inteligente de etiquetas HTML y CSS
  • Prettier - Formatea tu código automáticamente para que sea más legible

Sitios útiles para consultar

  • MDN Web Docs - La documentación oficial más completa y confiable
  • W3Schools - Referencias rápidas y ejemplos interactivos
  • CanIUse - Verifica qué características son compatibles con cada navegador
  • HTML CheatSheet - Guías rápidas con todas las etiquetas HTML

Estructura básica de un archivo HTML

Todo documento HTML tiene una estructura base que siempre se repite. Veamos cómo se ve un archivo HTML básico:

Plantilla básica HTML HTML 
<!DOCTYPE html>
<html lang="es">

<head>
    <!-- Declaración de metadatos del documento -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi primera página</title>
</head>

<body>
    <!-- Todo el contenido visible irá aquí -->
    <h1>Hola, mundo</h1>
</body>

</html>

No te preocupes por los detalles ahora

En la siguiente lección explicaremos línea por línea qué significa cada parte de este código. Por ahora solo familiarízate con cómo se ve un documento HTML.

Datos interesantes sobre HTML

  • El primer sitio web de la historia aún existe y sigue en línea desde 1991
  • HTML originalmente tenía solo 18 etiquetas. Hoy hay más de 100
  • Antes de HTML5, necesitabas plugins como Flash para reproducir videos en web
  • Tim Berners-Lee también inventó el concepto de URL y el primer navegador web
  • HTML5 permite crear videojuegos completos que corren directamente en el navegador
  • La web moderna combina HTML + CSS + JavaScript, pero HTML es siempre el inicio

Actividad práctica: Crea tu primer archivo HTML

Es momento de poner en práctica lo aprendido. Esta actividad es simple y rápida, solo para que te familiarices con el proceso:

  1. Crea una carpeta en tu computadora llamada mi-primer-sitio
  2. Abre Visual Studio Code y abre esa carpeta
  3. Crea un nuevo archivo llamado index.html
  4. Escribe la plantilla básica de HTML que viste anteriormente
  5. Cambia el texto del h1 por tu nombre
  6. Guarda el archivo y ábrelo con tu navegador

Consejo

Para abrir el archivo en el navegador, puedes hacer clic derecho sobre index.html y seleccionar "Abrir con" o simplemente arrastrarlo a una ventana del navegador.

Resumen de la lección

En esta primera lección aprendiste que HTML es el lenguaje de marcado que estructura todas las páginas web. No es un lenguaje de programación, sino un sistema de etiquetas que organiza el contenido.

Conociste la historia de HTML, desde su creación por Tim Berners-Lee en 1991 hasta HTML5, el estándar actual que trajo importantes mejoras como etiquetas semánticas y soporte nativo para multimedia.

Identificaste las herramientas esenciales: Visual Studio Code como editor, Google Chrome con sus DevTools para inspeccionar páginas, y recursos como MDN Web Docs y W3Schools para consultar.

Finalmente, viste por primera vez la estructura básica de un documento HTML. En la próxima lección profundizaremos en cada elemento de esta estructura para que comprendas exactamente qué hace cada línea de código.
